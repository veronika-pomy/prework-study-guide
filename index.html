HTML

  • HTML stands for HyperText Markup Language
  • HTML provides a means to create structured documents using semantics for text such as headings, paragraphs, lists, and links
  • Tags are written using angle brackets
  • "Title" tag defines the browser page title shown on browser tabs
  • The start tag may also include the element's attributes within the tag
  • Some elements, such as the line break, "br", don't permit any embedded content
  • HTML headings are defined with "h1" to "h6", h1 being the most important to h6 being the least
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Comments are useful in understanding code and don't display on the webpage

CSS

  • CSS defines the look and layout of content
  • Each CSS rule consists of one or more selectors, and a declaration block
  • Selectors declare which part of the markup a style applies to by matching tags and attributes in the markup
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • id: an identifier unique within the document
  • class: an identifier that can annotate multiple elements in a document

Git

  • Git is a free and open source distributed version control system
  • Git was originally authored by Linus Torvalds in 2005
  • git init: creates an empty Git repository
  • git status: checks what branch we are currently on
  • git add -A: stages changes before committing
  • git commit -m “add a message”: commits new files/changes to the local repository
  • git remote -v: verifies local and remote repos are linked
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript affects behavior and content of web pages
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Conditional statements are used to perform different actions based on different conditions
  • Use "if" to specify a block of code to be executed, if a specified condition is true
  • Use "else" to specify a block of code to be executed, if the same condition is false
  • Use "else if" to specify a new condition to test, if the first condition is false
  • Use "switch" to specify many alternative blocks of code to be executed
  • Loops offer a quick and easy way to do something repeatedly
  • A function is a block of code designed to perform a particular task
  • A function needs to be "called" to be executed
